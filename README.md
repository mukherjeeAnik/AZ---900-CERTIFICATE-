## About the Certification

The [AZ-900: Microsoft Azure Fundamentals](https://learn.microsoft.com/en-us/credentials/certifications/azure-fundamentals/) certification validates foundational knowledge of cloud concepts and Microsoft Azure services.

**Issued by:** Microsoft  
**Credential ID:** *(add your credential ID here)*  
**Verify:** [Microsoft Credentials](https://learn.microsoft.com/en-us/users/me/credentials)

---

## Skills Validated

- **Cloud Concepts** — cloud computing models (IaaS, PaaS, SaaS), CapEx vs OpEx, shared responsibility model
- **Azure Architecture** — regions, availability zones, resource groups, subscriptions
- **Core Azure Services** — compute, networking, storage, and database services
- **Azure Management** — Azure Portal, CLI, ARM templates, Azure Advisor
- **Security & Compliance** — Microsoft Defender, Azure Policy, RBAC, Zero Trust
- **Pricing & Governance** — pricing calculator, TCO calculator, SLAs, service lifecycle

## What You Can Learn Next

AZ-900 is the entry point into the Azure certification ecosystem. Here's the full roadmap of skills and certifications you can pursue after completing it, organized by role and level.

### 🔵 Fundamentals (Same Level — Broaden Your Base)

| Certification | Focus |
|---|---|
| **AI-900** — Azure AI Fundamentals | AI concepts, Cognitive Services, responsible AI, Machine Learning basics |
| **DP-900** — Azure Data Fundamentals | Relational and non-relational data, analytics workloads, data roles |
| **SC-900** — Security, Compliance & Identity Fundamentals | Zero Trust, compliance frameworks, Microsoft Defender basics |

---

### 🟡 Associate Level (1–2 Years Experience Expected)

| Certification | Role | Skills Gained |
|---|---|---|
| **AZ-104** — Azure Administrator | Cloud Administrator | Managing VMs, storage, networking, RBAC, monitoring, and backups |
| **AZ-204** — Azure Developer | Cloud Developer | Building and deploying cloud apps, serverless, APIs, Cosmos DB, Azure Functions |
| **AZ-500** — Azure Security Engineer | Security Engineer | Identity & access management, security controls, threat protection, data security |
| **DP-203** — Azure Data Engineer | Data Engineer | Data pipelines, Azure Data Factory, Data Lake, Synapse Analytics |
| **DP-300** — Azure Database Administrator | Database Administrator | Managing SQL databases, performance tuning, high availability on Azure |

---

### 🔴 Expert Level (Deep Experience Required)

| Certification | Role | Skills Gained |
|---|---|---|
| **AZ-305** — Azure Solutions Architect | Solutions Architect | Designing end-to-end cloud solutions, hybrid architectures, service trade-off analysis |
| **AZ-400** — Azure DevOps Engineer | DevOps Engineer | CI/CD pipelines, infrastructure as code, automation, GitHub Actions, Azure DevOps |

---

### 🟣 Specialty (Niche Technical Domains)

| Certification | Focus |
|---|---|
